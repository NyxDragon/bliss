Help Guide

General Commands
+-------+----------------------------------------------------------------------+
|cls    | Clear terminal                                                       |
|calc   | Commissions price calculator                                         |
|conv   | Convert files to mp4 in convert folder                               |
|del    | Deletes file                                                         |
|edit   | Launch BLEdit software                                               |
|exit   | Ends BLISS session                                                   |
|help   | Displays help guide OR input help [command] for specific help        |
|info   | Displays current build information                                   |
|list   | Lists current directory contents                                     |
|man    | Launches BLISS manual viewer                                         |
|mdir   | Make directory at file path                                          |
|outf   | Display file contents to terminal                                    |
|rc     | BLISS Recompiling tool (Recompiles RC)                               |
|rdir   | Removes directory at file path                                       |
+-------+----------------------------------------------------------------------+
